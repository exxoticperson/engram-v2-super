# ENGRAM Case Study Forge

Use this skill to convert proof into leverage.

## Best use cases

- turning a pilot into a one-page case study
- building anonymized proof assets
- summarizing before-and-after improvements
- extracting outreach angles from audit outcomes

## Required ingredients

- client context
- problem before ENGRAM
- what ENGRAM found
- what changed
- what happened after implementation, if known

## Output formats

- one-page case study
- short proof blurb
- proposal slide
- LinkedIn post
- sales email snippet

## Rules

- never fabricate outcome data
- if data is partial, say so clearly
- lead with the problem and the insight, not brand hype
- keep the case study easy to forward internally
