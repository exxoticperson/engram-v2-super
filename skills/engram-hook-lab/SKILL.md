# ENGRAM Hook Lab

Use this skill to produce sharper openings and better first-impression logic.

## Use cases

- repair briefs inside audits
- sample audit hook rewrites
- outreach message openings
- LinkedIn content hooks
- landing-page hero language

## Hook types to explore

- problem-first
- curiosity-first
- social-proof-first
- direct-claim
- category-contrast
- identity-led

## Output requirements

Generate options that vary by:

- angle
- tone
- clarity
- aggression level
- buyer awareness

## Rules

- do not produce shallow variations that all say the same thing
- prefer specific, buyer-legible language
- optimize first for clarity, then novelty
- for audit work, connect hooks to the exact failure they are fixing

## Good output

The user should receive options that feel testable, not generic.
