# ENGRAM Trend Research

This is an internal analyst workflow. It supports the audit. It is not the public headline of the business.

## Purpose

Trend research gives the audit current category context so the diagnosis is not blind to what buyers are seeing in-market right now.

## Use this workflow before an audit

Primary sources:

- TikTok Creative Center
- Meta Ad Library
- structured manual observation
- optional real-time social research if available

## Output requirements

Every trend snapshot must include:

- week of analysis
- platform
- category
- data source label
- what is winning
- what is oversaturated
- what that means for the creative being audited

## Data-source rules

- If using real-time sources, label as Real-time
- If using only observed patterns and prior benchmark knowledge, label as Benchmark-Based
- Do not imply current-week certainty if the source base does not justify it

## Snapshot structure

- dominant hook style
- reveal timing sweet spot
- audio pattern
- visual pacing pattern
- CTA pattern
- oversaturated patterns
- Cultural-Neural Alignment interpretation

## Sample audit sourcing

Preferred categories:

- beauty
- skincare

Preferred sources:

- publicly visible ads in TikTok Creative Center
- publicly visible ads in Meta Ad Library

Use public content for analysis only. Deliver the analysis, not raw redistributed ad files.
