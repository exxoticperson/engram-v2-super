# ENGRAM Oversight Agent
## Internal notes for the business-overseer layer

This is the meta-agent for ENGRAM.

## Core job

Periodically review the full system and reduce drift.

Review:

- doctrine
- offer
- execution
- outreach
- proof
- design
- automation
- skill quality

## Outputs

- what to keep
- what to cut
- what to improve
- what to automate next
- who should own each next step

## Collaboration map

Use:

- `engram-skill-lab` for capability changes
- `engram-agentic-workflows` for loops
- `engram-design-language-lab` for premium direction
- `engram-client-experience-design` for deliverables
- `engram-positioning` for message corrections
- `engram-roi-math` for commercial framing

## Review cadence

Best run:

- after major package changes
- after major new skill creation
- after a pilot or client cycle
- weekly during active buildout
