# ENGRAM Client Experience Design

Use this skill to turn ENGRAM outputs into premium client experiences.

## Mission

Make ENGRAM feel like a world-class operator product in every client touchpoint.

## Formats this skill covers

- audit report layout
- report cover systems
- debrief slide structure
- onboarding packets
- proposal decks
- intake form UX
- client portal or dashboard concepts
- case-study layout
- sales one-pagers
- executive summaries

## What this skill should optimize

- clarity under time pressure
- premium perception
- trust and seriousness
- ease of forwarding internally
- elegance without fluff
- high signal per page or screen

## Design principles

- information first
- hierarchy must be obvious
- motion must be purposeful
- visuals should reinforce trust, not decoration
- every artifact should feel like it belongs to the same system

## Output format

For each artifact, produce:

- design goal
- audience
- layout direction
- section order
- component recommendations
- visual style notes
- motion notes if relevant
- implementation notes if relevant

## Collaboration

Use alongside:

- `engram-design-language-lab` for the macro design language
- `engram-brand-trust-audit` for credibility review
- `engram-audit` for report structure

## Assets

Use:

- `assets/design/artifact_design_backlog.csv`
- `assets/design/design_direction_brief.txt`
- `client-facing/ENGRAM_SAMPLE_AUDIT_TEMPLATE.txt`

## Rule

This skill should improve the user experience of ENGRAM deliverables, not just make them prettier.
