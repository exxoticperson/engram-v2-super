# ENGRAM ROI Math

Use this skill when the user needs numbers that sell without pretending to predict outcomes.

## What this skill does

- estimates avoided creative waste
- translates bad launch decisions into budget impact
- frames the value of pre-launch diagnosis
- supports proposals, calls, and case studies

## Inputs to use

- spend level
- ad count
- launch cadence
- estimated bad creative rate
- known hook or hold issues
- implementation rate if known

## Output format

Keep it simple:

- assumption set
- conservative estimate
- moderate estimate
- one-line commercial takeaway

## Rules

- use ranges, not false precision
- label assumptions clearly
- do not imply guaranteed revenue lift
- anchor value in reduced waste and better launch selection

## Best uses

- sales calls
- proposals
- pricing defense
- case-study framing
