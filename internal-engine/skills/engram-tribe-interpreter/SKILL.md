# ENGRAM TRIBE Interpreter

This is an internal R&D skill. It does not change the public positioning of ENGRAM.

## Purpose

Use this skill to translate model output into:

- zone-level signals
- diagnostic hints
- candidate metric flags
- internal response curves

## Hard rules

- do not expose model names in public-facing copy
- do not present model output as guaranteed outcome prediction
- use model output as analyst support, not as a replacement for judgment

## Internal scope

This skill may cover:

- local setup
- signal extraction
- rough ROI mapping
- experimental thresholding
- response-curve generation
- known limitations

## Delivery rule

Any client-facing output generated downstream must still use the canonical proof policy and the live ENGRAM language from the root package.
